# ProjetoProgramaDeVendas
Estou acostumado a usar um programa de vendas em código fechado na farmácia onde trabalho. O programa codigo aberto que fiz: soma o valor dos produtos de vendas, marca produtos nas contas dos clientes, cadastra produtos e clientes.

Este programa foi escrito usando python 3.13.5. principalmente em sistema GNU/Linux. 
Os requerimentos do programa e dependências estão no arquivo requeriments.txt, voçê pode usar o comando "pip install -R requeriments.txt" no diretório raiz do programa. Para executar o programa abra o arquivo main.py na pasta bin/.
Este projeto recebera atualizações e funcionalidades acicionais no futuro.
